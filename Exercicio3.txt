Exercicio 3

3-> Descubra a lógica e complete o próximo elemento:

a-> 1, 3, 5, 7, ___
b-> 2, 4, 8, 16, 32, 64, ____
c-> 0, 1, 4, 9, 16, 25, 36, ____
d-> 4, 16, 36, 64, ____
e-> 1, 1, 2, 3, 5, 8, ____
f-> 2,10, 12, 16, 17, 18, 19, ____


a-> A lógica é que cada número é o resultado da adição de 2 com o número anterior. O próximo número seria 9.
b-> A lógica é que cada número é o resultado da multiplicação do número anterior por 2. O próximo número seria 128.
c-> A lógica é que cada número é o resultado do quadrado do seu índice na sequência (começando em 0). O próximo número seria 49,que é o resultado do quadrado do próximo índice na sequência (7).
d-> A lógica é que cada número é o resultado do quadrado do número ímpar correspondente na sequência 1, 3, 5, 7, 9, .... O próximo número seria 100, que é o quadrado do próximo número ímpar na sequência (11).
e-> A lógica é a sequência de Fibonacci, onde cada número é a soma dos dois anteriores. O próximo número seria 13.
f-> A lógica é que cada número é o próximo número inteiro maior que a média dos dois números anteriores. O próximo número seria 20.
